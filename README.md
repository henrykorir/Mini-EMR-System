# Mini-EMR-System

## Overview
A simple system to manage patient registrations, clinician profiles, and visit tracking.

## Features
- Register patients
- Manage clinician profiles
- Track patient visits
- User authentication with JWT

## Technologies
- **Frontend**: ReactJS
- **Backend**: Express.js
- **Database**: MySQL
- **Authentication**: JWT
